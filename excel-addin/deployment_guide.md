# Руководство по запуску Excel Add-in (GitHub Pages)

Отличные новости! Ваш проект успешно загружен на GitHub Pages. Теперь серверная часть (HTML/JS/CSS) работает в интернете.

Осталось только "познакомить" Excel с вашим приложением через файл манифеста.

## 1. Проверка на GitHub

Сначала убедимся, что сайт доступен:

1. Зайдите в ваш репозиторий на GitHub.
2. Перейдите в **Settings** (Настройки) -> **Pages** (в меню слева).
3. Убедитесь, что в качестве источника (Source) выбрана ветка **gh-pages**.
4. Сверху должно быть написано: `Your site is live at https://dwukus.github.io/AI-assist-excel/`
   - _Примечание: Может потребоваться пара минут, прежде чем сайт заработает. Если видите ошибку 404, подождите немного._

## 2. Подготовка Манифеста

Специальный файл для _боевого_ режима уже создан.
Он находится у вас на компьютере по адресу:
`d:\Projects\VBA\excel-addin\dist\manifest.xml`

**Важно:** Этот файл уже настроен так, чтобы брать картинки и скрипты не с вашего компьютера, а с интернета (GitHub Pages).

## 3. Загрузка в Excel (Windows)

В настольной версии Excel для Windows самый надежный способ подключить свою надстройку — это "Общая папка" (Shared Folder). Это звучит сложно, но делается один раз.

### Шаг А: Создайте "Каталог"

1. Создайте любую папку на диске (например, `C:\MyExcelAddins`).
2. Скопируйте файл `d:\Projects\VBA\excel-addin\dist\manifest.xml` в эту папку.
3. **Самый важный шаг**: Нажмите правой кнопкой на папку `C:\MyExcelAddins` -> **Свойства** -> вкладка **Доступ** (Sharing) -> кнопка **Общий доступ...** (Share).
   - Выберите себя в списке, нажмите "Поделиться" и "Готово".
   - Запомните "Сетевой путь", который вам покажут (обычно это вида `\\ВАШ-КОМПЬЮТЕР\MyExcelAddins`).

### Шаг Б: Подключите в Excel

1. Откройте Excel.
2. Перейдите в **Файл** -> **Параметры** -> **Центр управления безопасностью** (Trust Center).
3. Нажмите **Параметры центра управления безопасностью...** (Trust Center Settings).
4. Слева выберите **Доверенные каталоги надстроек** (Trusted Add-in Catalogs).
5. В поле "URL-адрес каталога" вставьте тот самый сетевой путь из Шага А (например, `\\LOCALHOST\MyExcelAddins`).
   - _Хитрость: часто вместо имени компьютера можно написать `localhost`, то есть `\\localhost\MyExcelAddins`._
6. Нажмите **Добавить каталог**.
7. **Обязательно** поставьте галочку **Показывать в меню** (Show in Menu) напротив добавленного пути.
8. Нажмите ОК два раза.
9. **Перезапустите Excel**.

### Шаг В: Запуск

1. Откройте пустую книгу.
2. Вкладка **Вставка** (Insert) -> **Мои надстройки** (My Add-ins).
3. Сверху должна появиться вкладка **ОБЩАЯ ПАПКА** (SHARED FOLDER).
4. Там вы увидите **AI Excel Assistant**. Выберите его и нажмите "Добавить".

Теперь на вкладке "Главная" появится кнопка вашего ассистента, и он будет работать через интернет!

---

## Если вы используете Excel в браузере (Excel on the Web)

Тут всё проще:

1. Откройте Excel в браузере.
2. Вставка -> Надстройки.
3. Вкладка "Мои надстройки" -> Ссылка "Загрузить мою надстройку" (Upload My Add-in).
4. Выберите файл `d:\Projects\VBA\excel-addin\dist\manifest.xml`.
5. Готово!
